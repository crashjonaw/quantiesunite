## Quick Start

If you just want to get the site running:

1. Double-click `run.bat` in the BloomBurrow folder
2. Wait for two CMD windows to open (Flask + Cloudflare Tunnel)
3. Visit https://bloomburrow.sg

To stop the site, close both CMD windows.
